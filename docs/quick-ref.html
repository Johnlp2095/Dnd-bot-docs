🎯 Common Usage Patterns

Create a Character 
char = create_character("Hero", 1)
add_item(char, "Sword", 1)
add_spell(char, "Magic Missile")
Combat Round 
initialize_combat([hero, goblin])
result = make_attack(hero, goblin)
end_turn()
check_combat_end()
Roll Checks 
attack = roll_dice(1, 20, 5)
damage = roll_dice(2, 6, 3)
lucky_roll = roll_advantage(20)
Save Character 
save_character(hero)
all_chars = list_all_characters()
loaded = load_character("Hero")

📋 Common Parameters

Parameter Type Example Notes
level int 1-20 Character level in D&D
sides int 4, 6, 8, 10, 12, 20 Die size for rolling
modifier int -5 to +20 Bonus/penalty to rolls
ability_scores dict {"STR": 15, "DEX": 14} Character stats
hit_points int 1-100+ Current HP (max = max_hp)
ac int 10-20 Armor Class for defense
