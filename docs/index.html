Quick Start Guide

1. Basic Dice Roll

Example:

from dnd_toolkit import DiceRoller

# Roll 3d6 + 2
result = DiceRoller.roll_dice(3, 6, 2)
print(result["total"])  # Prints total including modifier

2. Create a Character

Example:

from dnd_toolkit import Character

# Create character
hero = Character("Aragorn", "player_id", "Ranger", level=5)
hero.abilities["strength"] = 16
hero.take_damage(5)  # Apply damage
hero.heal(10)        # Restore HP

3. Run Combat

Example:

from dnd_toolkit import Combat

# Start a battle
combat = Combat(player, enemy)
combat.roll_initiative()

# Attack
result = combat.perform_attack(player, enemy)
print(f"Damage: {result['damage']}")

# Next turn
combat.next_turn()

4. Save a Character

Example:

from dnd_toolkit import CharacterManager

# Save character
manager = CharacterManager()
manager.save_character(hero)

# Load character later
loaded = manager.load_character("Aragorn")

How Components Work Together

Key Insight: Components are independent but designed to work together seamlessly.

Example Workflow: Combat Encounter

Here's how components combine in a typical combat scenario:

  1. Character System - Create player and enemy characters
  2. Combat System - Initialize combat between them
  3. Dice Roller - Roll d20 for initiative
  4. Game Rules - Calculate proficiency bonuses
  5. Combat System - Perform attacks using dice rolls
  6. Character System - Apply damage to target
  7. Spell System - Cast spells if needed
  8. Database System - Save character progress
