🎲 Dice Roller Component

Core dice rolling engine for D&D calculations

Overview

The Dice Roller component handles all randomness in the D&D system. It supports rolling any die (d4, d6, d8, d10, d12, d20) with modifiers, advantage, and disadvantage.

📖 Key Concepts:
  • Dice Notation: "3d6+2" means roll 3 six-sided dice and add 2
  • Advantage: Roll twice, take the higher result
  • Disadvantage: Roll twice, take the lower result
  • Modifier: Bonus or penalty added to the total

Methods

roll_single_die(sides)

Roll a single die with the specified number of sides.

DiceRoller.roll_single_die(sides: int) → int

Parameters:

sides int
Number of sides on the die. Common values: 4, 6, 8, 10, 12, 20

Returns:

int: Random number between 1 and sides (inclusive)

Example:

# Roll a d20 (1-20)
result = DiceRoller.roll_single_die(20)
print(f"Rolled: {result}")  # Output: Rolled: 17

# Roll a d6
result = DiceRoller.roll_single_die(6)
print(result)  # Output: 3

Related Methods:

  • roll_dice() - Roll multiple dice
  • roll_advantage() - Roll with advantage

roll_dice(num_dice, sides, modifier=0)

Roll multiple dice and sum them together with an optional modifier.

DiceRoller.roll_dice(num_dice: int, sides: int, modifier: int = 0) → dict

Parameters:

num_dice int
How many dice to roll
sides int
Number of sides on each die
modifier int
Optional bonus or penalty (default: 0)

Returns:

dict: Dictionary containing:
  • "rolls" - List of individual die results
  • "total" - Sum including modifier
  • "modifier" - The modifier used
  • "notation" - Standard D&D notation string (e.g., "3d6+2")

Example: Attack Roll

# Roll d20 + 5 (for an attack with +5 modifier)
result = DiceRoller.roll_dice(1, 20, 5)
print(f"Attack roll: {result['total']}")
# Output: Attack roll: 19

# Example output dictionary:
# {
#     "rolls": [14],
#     "total": 19,
#     "modifier": 5,
#     "notation": "1d20+5"
# }

Example: Damage Roll

# Roll 2d6 + 3 (for weapon damage)
result = DiceRoller.roll_dice(2, 6, 3)
print(f"Damage: {result['total']}")
# Possible output: Damage: 12

# Example output dictionary:
# {
#     "rolls": [5, 4],
#     "total": 12,
#     "modifier": 3,
#     "notation": "2d6+3"
# }

Used In:

  • Combat System - For attack and damage rolls
  • Character System - For ability checks
  • Spell System - For spell damage

roll_advantage(num_sides=20)

Roll with advantage (roll twice, take the higher result). Common in D&D 5e when conditions favor you.

DiceRoller.roll_advantage(num_sides: int = 20) → tuple

Parameters:

num_sides int
Die size (default: 20 for a d20)

Returns:

tuple: (result, [roll1, roll2])
  • First element: The highest of the two rolls
  • Second element: List of both individual rolls

Example: Attack with Advantage

# You have advantage on this attack
result, both = DiceRoller.roll_advantage()
print(f"Advantage rolls: {both}")  # e.g., [8, 15]
print(f"Using: {result}")          # Takes the 15

# With a modifier
strength_mod = 3
total = result + strength_mod
print(f"Attack total: {total}")    # 15 + 3 = 18

Related:

  • roll_disadvantage() - The opposite of advantage
  • Combat System attack() - Uses advantage/disadvantage

roll_disadvantage(num_sides=20)

Roll with disadvantage (roll twice, take the lower result). Applied when conditions work against you.

DiceRoller.roll_disadvantage(num_sides: int = 20) → tuple

Parameters:

num_sides int
Die size (default: 20 for a d20)

Returns:

tuple: (result, [roll1, roll2])
  • First element: The lowest of the two rolls
  • Second element: List of both individual rolls

Example: Attack with Disadvantage

# You have disadvantage (e.g., prone)
result, both = DiceRoller.roll_disadvantage()
print(f"Disadvantage rolls: {both}")  # e.g., [12, 7]
print(f"Using: {result}")             # Takes the 7

# Apply strength modifier
total = result + 3
print(f"Attack total: {total}")       # 7 + 3 = 10

Common Use Cases

Attack Roll

# Fighter attacks with +4 bonus
result = DiceRoller.roll_dice(1, 20, 4)
if result["total"] >= enemy_ac:
    damage = DiceRoller.roll_dice(1, 8, 3)  # Weapon damage
    print(f"HIT! Damage: {damage['total']}")

Ability Check

# Strength check with modifier
strength_mod = (strength_score - 10) // 2
result = DiceRoller.roll_dice(1, 20, strength_mod)
print(f"You rolled: {result['total']}")

Initiative

# Roll for who goes first
dex_mod = 3  # Dexterity modifier
init_roll = DiceRoller.roll_dice(1, 20, dex_mod)
print(f"Initiative: {init_roll['total']}")

Saving Throw

# Make a Dexterity save
dex_save_mod = 4
result = DiceRoller.roll_dice(1, 20, dex_save_mod)
dc = 15  # Difficulty class
if result["total"] >= dc:
    print("You succeed!")
else:
    print("You fail!")

Tips & Tricks

Tip 1: Check Roll Range

When debugging, always verify your dice rolls are in the valid range. A d20 should always be 1-20 before modifiers.

Tip 2: Recalculate Modifiers

Don't hard-code modifiers! Calculate them from ability scores: 
strength_mod = (strength_score - 10) // 2
attack = DiceRoller.roll_dice(1, 20, strength_mod)

Tip 3: Advantage/Disadvantage Impact

Advantage increases your average roll by ~4.5, disadvantage decreases it by ~4.5. This is significant!

Tip 4: Storing Results

Store the full result dictionary, not just the total. You might need the individual rolls for logging or debugging. 
result = DiceRoller.roll_dice(3, 6, 2)
log_entry = f"{result['notation']} = {result['rolls']} + {result['modifier']} = {result['total']}"
print(log_entry)
