💾 Database System Component

Save and load characters to JSON files for persistent storage

Overview

The Database System handles character persistence by saving character data to JSON files and loading them back. This allows players to save progress between sessions.

📖 What Gets Saved:
  • Character name and class
  • All ability scores
  • Current and max HP
  • Armor class and proficiency
  • Experience points
  • Spells known
  • Inventory items

Core Methods

__init__(save_folder="characters")

Initialize the character manager with a folder for storing files.

CharacterManager(save_folder: str = "characters")

Example:

# Use default folder
manager = CharacterManager()

# Use custom folder
manager = CharacterManager("my_characters")

# Use guild-specific folder
manager = CharacterManager(f"characters_guild_{guild_id}")

save_character(character)

Save a character to a JSON file in the save folder.

manager.save_character(character: Character)

Example:

manager = CharacterManager()
hero = Character("Aragorn", "player1", "Ranger", level=5)

# Modify character
hero.abilities["strength"] = 16
hero.armor_class = 15

# Save to file
manager.save_character(hero)
print("Saved!")  # Creates: characters/aragorn.json

Related:

  • Character.to_dict() - Converts character to dict

load_character(character_name)

Load a character from a JSON file.

manager.load_character(character_name: str) → Character or None

Returns:

Character: Loaded character object
None: If character file not found

Example:

manager = CharacterManager()

# Load a saved character
hero = manager.load_character("Aragorn")

if hero:
    print(f"Loaded {hero.name}")
    print(f"Level {hero.level} {hero.char_class}")
    print(f"HP: {hero.current_hp}/{hero.max_hp}")
else:
    print("Character not found")

get_all_characters()

Get a list of all saved character names.

manager.get_all_characters() → list

Returns:

list: List of character names (without .json extension)

Example:

manager = CharacterManager()

# Get all saved characters
characters = manager.get_all_characters()

print(f"Saved characters: {characters}")
# Output: Saved characters: ['aragorn', 'gandalf', 'legolas']

# Display them
for char_name in characters:
    char = manager.load_character(char_name)
    print(f"  {char.name} (Level {char.level} {char.char_class})")

delete_character(character_name)

Delete a saved character file.

manager.delete_character(character_name: str) → bool

Returns:

bool: True if deleted, False if file not found

Example:

manager = CharacterManager()

if manager.delete_character("OldCharacter"):
    print("Character deleted")
else:
    print("Character not found")

File Format (JSON)

Characters are saved as JSON files with this structure:

Example: aragorn.json

{
  "name": "Aragorn",
  "player_name": "player123",
  "char_class": "Ranger",
  "level": 5,
  "abilities": {
    "strength": 16,
    "dexterity": 14,
    "constitution": 14,
    "intelligence": 12,
    "wisdom": 14,
    "charisma": 12
  },
  "current_hp": 10,
  "max_hp": 10,
  "armor_class": 15,
  "experience": 6500,
  "spells": ["Hunter's Mark", "Mending"],
  "inventory": [
    {"name": "Longsword", "quantity": 1},
    {"name": "Bow", "quantity": 1},
    {"name": "Arrow", "quantity": 20}
  ]
}
💡 Tip: JSON files are text-based and human-readable. You can edit them directly if needed!

Complete Save/Load Example

Full Session Workflow:

from dnd_toolkit import Character, CharacterManager

# Session start: Load previous characters
manager = CharacterManager("my_campaign")

# Load existing character
hero = manager.load_character("Aragorn")

if hero:
    print(f"Welcome back, {hero.name}!")
    print(f"HP: {hero.current_hp}/{hero.max_hp}")
else:
    # Create new character
    hero = Character("Aragorn", "player1", "Ranger", level=1)
    hero.abilities["strength"] = 14
    hero.abilities["dexterity"] = 16

# Play session...
hero.take_damage(5)
hero.add_item("Gold", 50)
hero.experience += 300

# Session end: Save progress
manager.save_character(hero)
print(f"Saved {hero.name}!")

# Load all characters for a summary
all_chars = manager.get_all_characters()
print(f"\nCampaign characters: {len(all_chars)}")
for name in all_chars:
    char = manager.load_character(name)
    print(f"  - {char.name} (Level {char.level})")

Discord Bot Integration

Bot Commands with Database:

import discord
from discord.ext import commands
from dnd_toolkit import Character, CharacterManager

bot = commands.Bot(command_prefix="!")
managers = {}  # One per guild

def get_manager(guild_id):
    if guild_id not in managers:
        managers[guild_id] = CharacterManager(f"characters_{guild_id}")
    return managers[guild_id]

@bot.command(name="save")
async def save_char(ctx, *, char_name):
    """Save your character"""
    manager = get_manager(ctx.guild.id)
    
    # Get character (from game state)
    char = active_characters.get(ctx.author.id)
    
    if char:
        manager.save_character(char)
        await ctx.send(f"Saved {char.name}!")
    else:
        await ctx.send("No active character to save")

@bot.command(name="load")
async def load_char(ctx, *, char_name):
    """Load a saved character"""
    manager = get_manager(ctx.guild.id)
    
    char = manager.load_character(char_name)
    if char:
        active_characters[ctx.author.id] = char
        await ctx.send(char.display_stats())
    else:
        await ctx.send(f"Character '{char_name}' not found")

@bot.command(name="characters")
async def list_chars(ctx):
    """List all saved characters"""
    manager = get_manager(ctx.guild.id)
    
    chars = manager.get_all_characters()
    if chars:
        msg = "**Saved Characters:**\n"
        for name in chars:
            char = manager.load_character(name)
            msg += f"  • {char.name} (Level {char.level} {char.char_class})\n"
        await ctx.send(msg)
    else:
        await ctx.send("No saved characters yet")
