✨ Spell System Component

Manage spellcasting, spell slots, and daily limits

Overview

The Spell System manages D&D spells including spell levels, damage, casting limits, and daily resets. Spells can be tracked by how many times they can be cast per day.

📖 Spell Types:
  • Cantrips (Level 0): Unlimited casts per day
  • Leveled Spells: Limited by spell slots (fewer as level increases)
  • Casting Ability: Intelligence (Wizards), Wisdom (Clerics), Charisma (Warlocks)

Core Methods

__init__(name, spell_level, damage_dice=1, damage_die=6, description="")

Create a new spell with its properties.

Spell(name: str, spell_level: int, damage_dice: int = 1, damage_die: int = 6, description: str = "")

Example:

# Fireball (3rd level spell, 8d6 damage)
fireball = Spell("Fireball", 3, 8, 6, "Huge explosion of fire")

# Magic Missile (1st level, 1d4 damage)
missile = Spell("Magic Missile", 1, 1, 4, "Magical projectiles")

# Fire Bolt (Cantrip, 1d10 damage)
bolt = Spell("Fire Bolt", 0, 1, 10, "A bolt of fire")

can_cast()

Check if the spell has uses remaining today.

spell.can_cast() → bool

Example:

fireball = Spell("Fireball", 3, 8, 6)

if fireball.can_cast():
    # Cast the spell
    fireball.cast()
else:
    print("Out of spell slots!")

cast()

Use one spell slot/use and reduce uses remaining.

spell.cast() → bool

Returns:

bool: True if successfully cast, False if out of uses

Example:

fireball = Spell("Fireball", 3, 8, 6)
print(f"Uses: {fireball.current_uses}/{fireball.uses_per_day}")  # Uses: 2/2

# Cast once
if fireball.cast():
    print("Fireball cast!")
    print(f"Uses: {fireball.current_uses}/{fireball.uses_per_day}")  # Uses: 1/2

# Cast again
if fireball.cast():
    print("Fireball cast!")
    print(f"Uses: {fireball.current_uses}/{fireball.uses_per_day}")  # Uses: 0/2

# Try to cast without uses
if not fireball.cast():
    print("No more uses!")

reset_daily()

Reset spell uses for a new day (long rest).

spell.reset_daily()

Example:

fireball = Spell("Fireball", 3, 8, 6)
fireball.cast()
fireball.cast()

print(f"Before rest: {fireball.current_uses}/{fireball.uses_per_day}")  # 0/2

# Long rest
fireball.reset_daily()
print(f"After rest: {fireball.current_uses}/{fireball.uses_per_day}")   # 2/2

roll_damage()

Roll damage dice for the spell.

spell.roll_damage() → int

Returns:

int: Total damage rolled

Example:

fireball = Spell("Fireball", 3, 8, 6)

# Roll damage (8d6)
damage = fireball.roll_damage()
print(f"Fireball damage: {damage}")

Uses:

  • Dice Roller - Rolls the damage dice
  • Combat System - In spell attacks

Spell Properties

Property Type Description
name str Spell name
spell_level int Spell level (0-9, 0 is cantrip)
damage_dice int Number of damage dice
damage_die int Type of damage die (6 for d6, 8 for d8, etc.)
uses_per_day int Daily use limit (unlimited for cantrips)
current_uses int Uses remaining today

Spell Database

Pre-defined spells available in the SPELL_DATABASE dictionary:

Example Spells:
  • Fire Bolt: Cantrip, 1d10 damage
  • Magic Missile: 1st level, 1d4 damage
  • Fireball: 3rd level, 8d6 damage
  • Healing Word: 1st level, 1d4 healing

Using Pre-defined Spells:

from dnd_toolkit import SPELL_DATABASE

# Get a pre-defined spell
fire_bolt = SPELL_DATABASE["fire_bolt"]

# Create your own spells
custom_spell = Spell("Custom Spell", 2, 3, 6)

Complete Spellcasting Example

Wizard with Spells:

from dnd_toolkit import Character, Spell

# Create a wizard
wizard = Character("Gandalf", "player1", "Wizard", level=5)
wizard.abilities["intelligence"] = 18

# Create and add spells
fireball = Spell("Fireball", 3, 8, 6)
magic_missile = Spell("Magic Missile", 1, 1, 4)
fire_bolt = Spell("Fire Bolt", 0, 1, 10)  # Cantrip

wizard.spells = [fireball, magic_missile, fire_bolt]

# Cast spells
print(f"Casting Fireball...")
if wizard.spells[0].can_cast():
    damage = wizard.spells[0].roll_damage()
    wizard.spells[0].cast()
    print(f"Damage: {damage}")
    print(f"Uses left: {wizard.spells[0].current_uses}/{wizard.spells[0].uses_per_day}")

# Cantrip has unlimited uses
print(f"\nCasting Fire Bolt (cantrip)...")
for i in range(5):
    damage = fire_bolt.roll_damage()
    fire_bolt.cast()
    print(f"  Cast {i+1}: {damage} damage")

print(f"Fire Bolt uses: {fire_bolt.current_uses}/{fire_bolt.uses_per_day}")
