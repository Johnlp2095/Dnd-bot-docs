👤 Character System Component

Complete character creation and management system for D&D

Overview

The Character System is the foundation for creating and managing player characters and NPCs. It stores all character data including stats, hit points, spells, inventory, and status effects.

📖 Character Data:
  • Abilities: Six core stats (STR, DEX, CON, INT, WIS, CHA) ranging 3-18
  • Hit Points: Current and maximum health
  • Armor Class: How hard to hit
  • Inventory: Equipment and items carried
  • Spells: Known spells and cantrips
  • Status Effects: Conditions like poison, stun, charm

Creating a Character

__init__(name, player_name, char_class, level=1)

Create a new character with basic information.

Character(name: str, player_name: str, char_class: str, level: int = 1)

Parameters:

name str
Character's name (e.g., "Aragorn")
player_name str
Discord ID or player identifier
char_class str
Character class: Fighter, Wizard, Rogue, Cleric, Ranger, Paladin, Barbarian, Bard, etc.
level int
Starting level (default: 1, max: 20)

Example:

# Create a level 5 Ranger named Aragorn
hero = Character("Aragorn", "player_123", "Ranger", level=5)

# Create a level 1 Wizard
mage = Character("Gandalf", "player_456", "Wizard", level=1)

Initial Character Stats

Ability Starting Value Purpose
Strength 10 Melee damage, carrying capacity
Dexterity 10 Dodge, initiative, ranged attacks
Constitution 10 Health, hit points
Intelligence 10 Wizard spellcasting
Wisdom 10 Cleric/Druid spellcasting, perception
Charisma 10 Persuasion, deception, leadership

Core Methods

get_ability_modifier(ability_name)

Calculate the modifier for a specific ability. Used in most rolls and checks.

character.get_ability_modifier(ability_name: str) → int

Parameters:

ability_name str
Name of ability: "strength", "dexterity", "constitution", "intelligence", "wisdom", "charisma"

Returns:

int: The modifier value
  • Score 10-11 → Modifier 0
  • Score 12-13 → Modifier +1
  • Score 14-15 → Modifier +2
  • Score 16-17 → Modifier +3
  • Score 18-19 → Modifier +4

Example:

hero = Character("Aragorn", "player1", "Ranger")
hero.abilities["strength"] = 16

# Get strength modifier
str_mod = hero.get_ability_modifier("strength")
print(f"Strength modifier: +{str_mod}")  # Output: Strength modifier: +3

# Use in attack roll
attack_roll = DiceRoller.roll_dice(1, 20, str_mod)

Used In:

  • Combat System - For attack calculations
  • Dice Roller - Adding to roll modifiers
  • Game Rules - For proficiency calculations

take_damage(damage)

Reduce character's HP when taking damage. Character dies if HP reaches 0.

character.take_damage(damage: int) → bool

Parameters:

damage int
Amount of damage to apply

Returns:

bool: True if character survives, False if defeated (HP ≤ 0)

Example:

hero = Character("Aragorn", "player1", "Ranger")
print(f"HP: {hero.current_hp}/{hero.max_hp}")  # HP: 10/10

# Take 5 damage
is_alive = hero.take_damage(5)
print(f"HP: {hero.current_hp}/{hero.max_hp}")  # HP: 5/10
print(f"Alive: {is_alive}")  # Alive: True

# Take lethal damage
is_alive = hero.take_damage(10)
print(f"HP: {hero.current_hp}/{hero.max_hp}")  # HP: 0/10
print(f"Alive: {is_alive}")  # Alive: False

Related Methods:

  • heal() - Restore HP
  • Combat System - Calls take_damage on targets

heal(amount)

Restore hit points. Cannot exceed maximum HP.

character.heal(amount: int)

Parameters:

amount int
HP to restore (e.g., from a potion or healing spell)

Example:

hero = Character("Aragorn", "player1", "Ranger")
hero.current_hp = 3

# Use a healing potion
hero.heal(5)
print(f"HP: {hero.current_hp}/{hero.max_hp}")  # HP: 8/10

# Try to over-heal (only restores to max)
hero.heal(10)
print(f"HP: {hero.current_hp}/{hero.max_hp}")  # HP: 10/10

add_spell(spell_name)

Add a known spell to the character's spell list.

character.add_spell(spell_name: str)

Parameters:

spell_name str
Name of the spell to learn

Example:

mage = Character("Gandalf", "player456", "Wizard", level=5)

# Add spells to the wizard
mage.add_spell("Fireball")
mage.add_spell("Magic Missile")
mage.add_spell("Mending")

print(f"Spells: {mage.spells}")
# Output: Spells: ['Fireball', 'Magic Missile', 'Mending']

Related:

  • Spell System - Full spell management
  • Combat System spell attacks

add_item(item_name, quantity=1)

Add an item to the character's inventory.

character.add_item(item_name: str, quantity: int = 1)

Parameters:

item_name str
Name of item to add
quantity int
How many of this item (default: 1)

Example:

hero = Character("Aragorn", "player1", "Ranger")

# Add equipment
hero.add_item("Sword")
hero.add_item("Bow")
hero.add_item("Arrow", 20)
hero.add_item("Healing Potion", 3)

print(hero.inventory)
# Output: [
#   {'name': 'Sword', 'quantity': 1},
#   {'name': 'Bow', 'quantity': 1},
#   {'name': 'Arrow', 'quantity': 20},
#   {'name': 'Healing Potion', 'quantity': 3}
# ]

add_status_effect(effect_name) / remove_status_effect(effect_name)

Manage status effects like poison, stun, charm, etc.

character.add_status_effect(effect_name: str)
character.remove_status_effect(effect_name: str)

Example:

hero = Character("Aragorn", "player1", "Ranger")

# Apply effects
hero.add_status_effect("poisoned")
hero.add_status_effect("blinded")
print(hero.status_effects)  # ['poisoned', 'blinded']

# Remove an effect
hero.remove_status_effect("blinded")
print(hero.status_effects)  # ['poisoned']

is_alive()

Check if character is still alive (HP > 0).

character.is_alive() → bool

Example:

hero = Character("Aragorn", "player1", "Ranger")

if hero.is_alive():
    print("Character is alive!")
else:
    print("Character has been defeated!")

display_stats()

Get a formatted character sheet for display.

character.display_stats() → str

Example:

hero = Character("Aragorn", "player1", "Ranger", level=5)
hero.abilities["strength"] = 16
hero.abilities["dexterity"] = 14

print(hero.display_stats())
# Output:
# **Aragorn** (Level 5 Ranger)
# HP: 10/10
# AC: 10
# 
# Abilities:
#   Strength: 16 (+3)
#   Dexterity: 14 (+2)
#   Constitution: 10 (+0)
#   Intelligence: 10 (+0)
#   Wisdom: 10 (+0)
#   Charisma: 10 (+0)

Useful For:

  • Discord bot responses
  • Displaying character information
  • Logging character state

to_dict()

Convert character to dictionary format for saving.

character.to_dict() → dict

Returns:

dict: Contains all character information

Used By:

  • Database System - To save characters to JSON

Character Properties

Property Type Description
name str Character's name
player_name str Player ID or Discord username
char_class str Character class (Fighter, Wizard, etc.)
level int Character's experience level
abilities dict Six ability scores: strength, dexterity, constitution, intelligence, wisdom, charisma
current_hp int Current hit points
max_hp int Maximum hit points
armor_class int How hard to hit (default: 10)
experience int Current experience points
spells list Known spells
inventory list Items in inventory
status_effects list Active conditions (poison, stun, etc.)

Common Workflows

Workflow 1: Create a Fighter

hero = Character("Conan", "player1", "Fighter", level=5)

# Fighters are strong and tough
hero.abilities["strength"] = 18
hero.abilities["constitution"] = 16
hero.armor_class = 16

# Give them weapons
hero.add_item("Longsword")
hero.add_item("Shield")
hero.add_item("Armor")

print(hero.display_stats())

Workflow 2: Combat Scenario

player = Character("Hero", "p1", "Fighter")
enemy = Character("Goblin", "npc1", "Rogue")

# Enemy attacks player
damage = 5
player.take_damage(damage)

if not player.is_alive():
    print(f"{player.name} has been defeated!")
else:
    print(f"{player.name} HP: {player.current_hp}/{player.max_hp}")

# Player gets healed
healing_potion = 8
player.heal(healing_potion)
print(f"Healed! HP: {player.current_hp}/{player.max_hp}")

Workflow 3: Setup Wizard

wizard = Character("Gandalf", "player2", "Wizard", level=10)

# Wizards are smart but fragile
wizard.abilities["intelligence"] = 18
wizard.abilities["dexterity"] = 12
wizard.armor_class = 12

# Add spells
wizard.add_spell("Fireball")
wizard.add_spell("Magic Missile")
wizard.add_spell("Mending")

# Equipment
wizard.add_item("Staff")
wizard.add_item("Spellbook")

print(f"{wizard.name} knows: {wizard.spells}")
