📖 Game Rules Component

D&D rules reference, calculations, and lookup tables

Overview

The Game Rules component provides D&D calculations and lookup tables including experience requirements, proficiency bonuses, class hit dice, and critical hit detection.

📖 What It Contains:
  • Experience point table for leveling
  • Class-specific hit dice
  • Proficiency bonus calculations
  • Critical hit/miss detection

Core Methods

get_proficiency_bonus(level)

Calculate proficiency bonus based on character level. Used in most attack and skill rolls.

GameRules.get_proficiency_bonus(level: int) → int

Parameters:

level int
Character level (1-20)

Returns:

int: Proficiency bonus
  • Levels 1-4: +2
  • Levels 5-8: +3
  • Levels 9-12: +4
  • Levels 13-16: +5
  • Levels 17-20: +6

Example:

for level in [1, 5, 10, 15, 20]:
    prof = GameRules.get_proficiency_bonus(level)
    print(f"Level {level}: +{prof}")

Used In:

  • Combat System - Added to attack rolls
  • Character System - Stored in proficiency_bonus

get_xp_needed(current_level)

Get experience points needed to reach the next level.

GameRules.get_xp_needed(current_level: int) → int

Example:

xp_for_level_2 = GameRules.get_xp_needed(1)  # 300
xp_for_level_3 = GameRules.get_xp_needed(2)  # 900
xp_for_level_5 = GameRules.get_xp_needed(4)  # 6500

print(f"Need {xp_for_level_2} XP to reach level 2")

Related:

  • Experience Table - Full XP requirements

get_hit_die(char_class)

Get the hit die (damage die) for a character class. Determines HP gain per level.

GameRules.get_hit_die(char_class: str) → int

Parameters:

char_class str
Character class name (case-insensitive)

Returns:

int: Number of sides on the hit die

Example:

hit_dice = {
    "barbarian": GameRules.get_hit_die("Barbarian"),    # 12
    "fighter": GameRules.get_hit_die("Fighter"),        # 10
    "rogue": GameRules.get_hit_die("Rogue"),            # 8
    "wizard": GameRules.get_hit_die("Wizard"),          # 6
}

for class_name, hit_die in hit_dice.items():
    print(f"{class_name} uses d{hit_die}")

Related:

  • Dice Roller - Rolls the hit die for HP
  • Class Hit Dice Table

is_critical_hit(d20_roll)

Check if a d20 roll is a critical hit (natural 20).

GameRules.is_critical_hit(d20_roll: int) → bool

Example:

roll = 20
if GameRules.is_critical_hit(roll):
    print("Critical hit! Double damage!")
    
roll = 19
if not GameRules.is_critical_hit(roll):
    print("Not quite...")

is_critical_miss(d20_roll)

Check if a d20 roll is a critical miss (natural 1).

GameRules.is_critical_miss(d20_roll: int) → bool

Example:

roll = 1
if GameRules.is_critical_miss(roll):
    print("Critical failure! Oh no!")
    
roll = 2
if not GameRules.is_critical_miss(roll):
    print("Better luck next time")

calculate_ac_from_dex_and_armor(base_ac, dex_modifier)

Calculate total AC from armor and dexterity modifier.

GameRules.calculate_ac_from_dex_and_armor(base_ac: int, dex_modifier: int) → int

Example:

# Leather armor (base AC 11) with +3 DEX
ac = GameRules.calculate_ac_from_dex_and_armor(11, 3)
print(f"AC: {ac}")  # AC: 14

# Heavy armor (base AC 17) - DEX doesn't apply fully
ac = GameRules.calculate_ac_from_dex_and_armor(17, 3)
print(f"AC: {ac}")  # AC: 17 (no DEX bonus)

Usage Examples

Example: Leveling Up

def level_up(character):
    """Level up a character"""
    character.level += 1
    
    # Update proficiency
    character.proficiency_bonus = GameRules.get_proficiency_bonus(character.level)
    
    # Increase HP
    hit_die = GameRules.get_hit_die(character.char_class)
    new_hp = DiceRoller.roll_single_die(hit_die)
    new_hp += character.get_ability_modifier("constitution")
    character.max_hp += max(1, new_hp)
    character.current_hp = character.max_hp
    
    return character

# Usage
hero = Character("Aragorn", "p1", "Fighter", level=1)
hero = level_up(hero)
print(f"Level up! Now level {hero.level} with {hero.max_hp} max HP")

Example: Check for Critical

def attack_with_crit(attacker, defender):
    """Perform attack and check for critical"""
    # Roll attack
    d20 = DiceRoller.roll_single_die(20)
    attack_roll = d20 + attacker.get_ability_modifier("strength")
    
    # Check for critical
    if GameRules.is_critical_hit(d20):
        print("CRITICAL HIT!")
        # Double damage
        damage = DiceRoller.roll_dice(2, 8, attacker.get_ability_modifier("strength"))
    elif GameRules.is_critical_miss(d20):
        print("CRITICAL FAILURE!")
        return
    elif attack_roll >= defender.armor_class:
        damage = DiceRoller.roll_dice(1, 8, attacker.get_ability_modifier("strength"))
    else:
        print("Miss!")
        return
    
    defender.take_damage(damage["total"])
