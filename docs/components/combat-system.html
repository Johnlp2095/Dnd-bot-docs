⚔️ Combat System Component

Turn-based combat with initiative, attacks, and spell casting

Overview

The Combat System manages turn-based battles between multiple characters. It handles initiative rolls, attack calculations, damage application, and tracks combat state through multiple rounds.

📖 Combat Flow:
  1. Initialize combat with participants
  2. Roll initiative for turn order
  3. Each turn: active character attacks
  4. Damage applied and HP tracked
  5. Next turn until combat ends

Core Methods

__init__(*characters)

Initialize a combat encounter with participating characters.

Combat(*characters: Character)

Example:

# Start combat between player and enemy
player = Character("Hero", "p1", "Fighter")
enemy = Character("Goblin", "npc1", "Rogue")

combat = Combat(player, enemy)

# Or with multiple combatants
team1 = [player1, player2]
team2 = [enemy1, enemy2]
combat = Combat(player1, player2, enemy1, enemy2)

roll_initiative()

Roll d20 + DEX modifier for each combatant to determine turn order.

combat.roll_initiative() → list

Returns:

list: List of initiative rolls sorted highest first

Example:

player = Character("Hero", "p1", "Fighter")
player.abilities["dexterity"] = 14

enemy = Character("Goblin", "npc1", "Rogue")
enemy.abilities["dexterity"] = 16

combat = Combat(player, enemy)
initiative = combat.roll_initiative()

for entry in initiative:
    print(f"{entry['character'].name}: {entry['roll']}")

Uses:

  • Dice Roller - Rolls d20 + DEX modifier
  • Character System - Gets DEX ability modifier

perform_attack(attacker, defender, weapon_damage=8)

Perform a melee attack roll and apply damage if it hits.

combat.perform_attack(attacker: Character, defender: Character, weapon_damage: int = 8) → dict

Parameters:

attacker Character
Character performing the attack
defender Character
Character being attacked
weapon_damage int
Weapon damage die (default: 8 for longsword)

Returns:

dict: Attack result with hit/miss, damage, and HP

Example:

combat = Combat(player, enemy)
combat.roll_initiative()

# Player attacks
result = combat.perform_attack(player, enemy, weapon_damage=8)

if result['hit']:
    print(f"HIT! {result['damage']} damage!")
else:
    print("MISS!")

print(f"Enemy HP: {result['defender_hp']}/{enemy.max_hp}")

Uses:

  • Dice Roller - Roll d20 and damage dice
  • Character System - Apply damage, get modifiers
  • Game Rules - Proficiency bonus

perform_spell_attack(caster, defender, spell_damage=6, spell_ability="intelligence")

Cast a spell that requires an attack roll and deals damage.

combat.perform_spell_attack(caster, defender, spell_damage, spell_ability) → dict

Example:

wizard = Character("Gandalf", "p2", "Wizard")
wizard.abilities["intelligence"] = 18

combat = Combat(wizard, enemy)

# Cast Fireball (2d6 damage, intelligence-based)
result = combat.perform_spell_attack(wizard, enemy, spell_damage=6, spell_ability="intelligence")

print(f"Spell Attack: {result['spell_attack']}")
if result['hit']:
    print(f"Damage: {result['damage']}")

  • Spell System - Full spell management

next_turn()

Move to the next combatant's turn and increment round counter.

combat.next_turn()

Example:

combat = Combat(player, enemy)
combat.roll_initiative()

# Combat round loop
while not combat.is_combat_finished():
    current_char = combat.get_current_combatant()
    print(f"{current_char.name}'s turn (Round {combat.round})")
    
    # Perform action...
    
    combat.next_turn()  # Move to next character

get_current_combatant()

Get the character whose turn it currently is.

combat.get_current_combatant() → Character

Example:

current = combat.get_current_combatant()
print(f"It's {current.name}'s turn!")

is_combat_finished()

Check if combat has ended (only one combatant alive).

combat.is_combat_finished() → bool

Returns:

bool: True if combat is over, False if still ongoing

Example:

if combat.is_combat_finished():
    print("Combat has ended!")
    # Determine victor
    alive = [c for c in combat.characters if c.is_alive()]
    if alive:
        print(f"{alive[0].name} wins!")

get_status()

Get formatted combat status for display.

combat.get_status() → str

Example:

print(combat.get_status())
# Output:
# **Round 3** - Hero's turn
#
# Hero: 12/20 HP
# Goblin: 5/8 HP

Combat Properties

Property Type Description
characters list All combatants in order of initiative
current_turn_index int Index of current combatant
round int Current round number (starts at 1)
history list Log of all actions taken

Complete Combat Example

Full Combat Encounter:

from dnd_toolkit import Character, Combat

# Create combatants
hero = Character("Hero", "p1", "Fighter", level=3)
hero.abilities["strength"] = 16
hero.armor_class = 15

enemy = Character("Goblin", "npc1", "Rogue", level=1)
enemy.abilities["dexterity"] = 14
enemy.armor_class = 13
enemy.max_hp = 7
enemy.current_hp = 7

# Start combat
combat = Combat(hero, enemy)
print("Combat started!")

# Roll initiative
init = combat.roll_initiative()
print("\nInitiative:")
for entry in init:
    print(f"  {entry['character'].name}: {entry['roll']}")

# Run combat rounds
round_num = 1
while not combat.is_combat_finished():
    print(f"\n--- Round {combat.round} ---")
    current = combat.get_current_combatant()
    print(f"{current.name}'s turn")
    
    # Determine opponent
    opponent = combat.characters[1] if combat.characters[0] == current else combat.characters[0]
    
    # Perform attack
    result = combat.perform_attack(current, opponent)
    print(f"Attack roll: {result['attack_roll']}")
    
    if result['hit']:
        print(f"HIT! {result['damage']} damage!")
    else:
        print("MISS!")
    
    print(combat.get_status())
    
    # End turn
    combat.next_turn()

# Determine winner
print("\n=== Combat Finished ===")
alive = [c for c in combat.characters if c.is_alive()]
if alive:
    print(f"{alive[0].name} is victorious!")
