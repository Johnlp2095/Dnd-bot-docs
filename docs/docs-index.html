🌐 GitHub Pages Hosting

Your documentation is hosted on GitHub Pages!

It's automatically served from the docs/ folder in your repository.

Steps to Activate:

  1. Push this documentation to your GitHub repository
  2. Go to Repository Settings → GitHub Pages
  3. Select: Branch = main, Folder = /docs
  4. Your site will be available at:
    https://yourusername.github.io/dnd-bot-copilot/docs

Features Included:

  • ✅ Automatic Jekyll builds
  • ✅ SEO optimization (sitemap.xml, robots.txt)
  • ✅ No build tools required
  • ✅ Custom domain support
  • ✅ SSL/HTTPS included
